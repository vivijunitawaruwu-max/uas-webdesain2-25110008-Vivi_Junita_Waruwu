## Deskripsi

Website Company Profile untuk **UMKM Soto Mak Uncu** yang dibuat sebagai tugas mata kuliah **Web Design 2 (MKK 123)**. Website ini menampilkan informasi mengenai UMKM, menu makanan, testimoni pelanggan, serta kontak dan pemesanan yang dikemas dalam tampilan modern, responsif, dan mudah digunakan.

## Fitur Website

### Header / Global
- Responsive Navbar menggunakan Bootstrap 5 (`navbar-expand-lg`).

### Beranda (Hero Section)
- Badges konten & informasi statistik:
  - 31+ Tahun Berdiri
  - 15K+ Pelanggan
  - 4.9 Rating Google
- Floating Badge status toko (Open 08.00–21.00).

### Tentang Kami
- Visi & Misi dalam bentuk card.
- Nilai jual utama (Features Badge).
- Profile singkat Founder / Head Chef (Mak Uncu).

### Menu (Products)
- E-commerce Grid Menu.
- Label khusus:
  - Best Seller
  - Premium
  - Hemat
  - Habis
- Tombol interaktif **Pesan** (`btn-add-cart`) menggunakan data attributes.

### Testimoni
- Sistem rating bintang.
- Kartu review pelanggan.

### Kontak
- Peta lokasi interaktif (Embedded Google Maps).
- Form pemesanan langsung terintegrasi WhatsApp.

### Footer
- Navigasi sekunder.
- Tautan media sosial.

### Sistem Animasi
- Integrasi library AOS (Animate On Scroll).

## Teknologi yang Digunakan

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- AOS (Animate On Scroll)
- Google Maps Embed
